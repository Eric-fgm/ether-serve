# Installation

<Steps>
### Install the Package

<Code>
  <NodeCode type="NodeCode">

    ```bash npm2yarn
    npm install @ether-serve/node
    ```

  </NodeCode>
</Code>

### Setup

Simple example of Hello World application.

<Code>
  <NodeCode type="NodeCode">
    ```ts filename="index.ts"
    import { ether, serve } from "@ether-serve/node";

    const app = ether();
    app.get("/", (c) => c.status(200).text("Hello World"));

    serve({
        fetch: app.fetch.bind(app),
        port: 8080,
    });
    ```

  </NodeCode>
</Code>

### Run

Run the development server locally.

<Code>
  <NodeCode type="NodeCode">

    ```bash npm2yarn
    npm run dev
    ```

  </NodeCode>
</Code>
