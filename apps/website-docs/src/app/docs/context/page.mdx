# Context

The Context object can be used to handle both the Request and Response.

### req

<Code>
  <NodeCode type="NodeCode">
    ```ts filename="index.ts"
    app.get('/', async (c) => {
        console.log(c.req.params()); // Parameters
        console.log(c.req.query()); // Query object
        console.log(await c.req.json()); // JSON body
        console.log(await c.req.text()); // Text body
        console.log(await c.req.body()); // Raw body
        console.log(c.req.protocol);
        console.log(c.req.url);
        console.log(c.req.path);
        console.log(c.req.pathname);
        console.log(c.req.pathRegex);
        console.log(c.req.method);
        console.log(c.req.headers);
        //...
    })
    ```

    </NodeCode>

</Code>
